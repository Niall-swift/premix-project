# AVL TELCOM - Website

Site institucional da AVL TELCOM, focado em alta tecnologia e conectividade premium.

## Tecnologias Utilizadas

- **Vite**: Build tool e servidor de desenvolvimento ultra-rápido.
- **Vanilla JS**: Lógica de animações e interações.
- **CSS3**: Design Responsivo, Glassmorphism e animações.
- **Canvas API**: Fundo interativo com partículas.

## Desenvolvimento

Para rodar o projeto localmente:

```bash
npm install
npm run dev
```

## Produção

Para gerar o build de produção:

```bash
npm run build
```

O comando acima irá gerar uma pasta `dist/` com todos os arquivos otimizados e prontos para serem hospedados em qualquer servidor estático (Vercel, Netlify, GitHub Pages, HostGator, etc.).

## Visualização de Produção

Para visualizar o build final localmente:

```bash
npm run preview
```
